# HTML : Quais elementos estão sendo usados?
O <h1> usei para criar o titulo do site.
A <div> funciona como um "divisor", eu usei para agrupar os elementos do site.
O <nav>, eu usei a class "navbar" para conectar os links de navegação dentro da barra.
Usei a tag <ul> para criar uma lista não ordenada.
Usei a tag <il> para estruyurar o conteúdo, e sempre entre as tags de abertura e fechamento de uma lista.
A tag <a> eu usei para adicionar p link na página.
O <link> eu usei para sincronizar o HTML como CSS, dessa maneira: <link rel="stylesheet" href="/css/style.css">.

# CSS : Quais propriedades estão sendo usadas?
margin
Define a margem externa do elemento (espaço entre o elemento e outros elementos).

margin: 0; remove qualquer margem padrão que navegadores adicionam a certos elementos (como <body>, <h1>, <p> etc.).
padding
Define o preenchimento interno do elemento (espaço entre o conteúdo e a borda).

padding: 0; remove o espaçamento interno padrão que alguns elementos possuem.
box-sizing
Define como o navegador deve calcular a largura e altura total de um elemento.
box-sizing: border-box; faz com que width e height incluam o padding e a border, o que facilita muito o controle do layout.

Define a cor de fundo do elemento.
No exemplo, o fundo será rosa (pink).

background-image -> Define uma imagem de fundo para o elemento.
